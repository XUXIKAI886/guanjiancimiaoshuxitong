# 快速开始指南

## 🚀 3步启动项目

### 步骤1: 安装依赖
```bash
cd product-optimizer
npm install
```

### 步骤2: 启动开发服务器
```bash
npm run dev
```

### 步骤3: 打开浏览器
访问 http://localhost:3000

---

## 📝 使用示例

### 关键词优化示例

**输入:**
```
卤鸭板肠(小份)
柠檬茶(大杯)
卤鸭板肠(小份)+卤菜(小份)
```

```

卤鸭板肠(小份)+卤菜(小份)【🍱组合丰富】
```

## 💡 使用技巧

1. **批量处理**: 一次可输入多个产品,每行一个
2. **套餐格式**: 使用 `+` 连接多个产品,如 `A+B+C`
3. **导出时机**: 生成结果后,可随时导出Excel或图片
4. **清空重置**: 点击"清空"按钮可重新开始

## ❓ 常见问题

**Q: API调用失败怎么办?**
A: 检查 `.env.local` 配置是否正确,确保API Key有效

**Q: 如何更换AI模型?**
A: 修改 `.env.local` 中的 `NEXT_PUBLIC_API_MODEL` 变量

**Q: 可以离线使用吗?**
A: 需要网络连接才能调用AI API
